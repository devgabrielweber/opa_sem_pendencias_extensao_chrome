Política de Privacidade – Opa! Sem pendências

Esta política de privacidade descreve como a extensão Opa! Sem pendências lida com dados durante o seu uso. Ao utilizar a nossa extensão, você concorda com os termos descritos nesta política.

1. Informações Coletadas

A extensão Opa! Sem pendências não coleta dados pessoais dos usuários. A única informação acessada é o valor do campo "Aguardando Resposta" de uma página específica que você está visualizando, a qual representa o número de atendimentos não respondidos. Esses dados são utilizados exclusivamente para exibir informações e alterar o conteúdo da página que você está visualizando.

2. Uso dos Dados

Os dados do campo "Aguardando Resposta" são lidos apenas para modificar dinamicamente a página e exibir um alerta (popup) caso o número de atendimentos não respondidos seja maior que zero. A extensão não armazena nem compartilha qualquer dado coletado, sendo utilizada somente para fins de modificação do conteúdo da página conforme o valor indicado.

3. Permissões Solicitadas

A extensão solicita as seguintes permissões:

  • Scripting: Para permitir a modificação do conteúdo HTML da página e inserir a página secundária onde as informações são extraídas.
  • Tabs: Para acessar a aba onde a extensão está sendo executada, a fim de modificar o conteúdo da página conforme os dados coletados.

Essas permissões são necessárias para o funcionamento da extensão e para proporcionar a experiência de uso descrita.

4. Armazenamento e Compartilhamento de Dados

A extensão não armazena, compartilha nem envia quaisquer dados para servidores externos ou terceiros. Todas as informações acessadas são processadas localmente na sua navegação e não são registradas ou transmitidas.

5. Segurança

Embora a extensão não colete dados sensíveis, garantimos que qualquer dado acessado é utilizado unicamente para modificar o conteúdo da página de forma segura e sem riscos à sua privacidade.

6. Alterações nesta Política de Privacidade

Se houver alguma alteração significativa nas práticas descritas nesta política, atualizaremos esta página e notificaremos os usuários conforme necessário. Recomendamos que você revise periodicamente nossa política de privacidade.
